# E-Waste Collection Locator

A responsive web application that helps users locate the nearest e-waste disposal centers.

## Features
- Google Maps integration
- Shows nearest e-waste drop-points
- User location detection
- Clean and mobile-responsive UI

## Tech Stack
HTML, CSS, JavaScript, Google Maps API

## How to Setup
1. Download the project
2. Replace YOUR_API_KEY in index.html with your Google Maps API key
3. Open index.html in browser

## Future Scope
- Add live API-based e-waste centers
- Add directions & distance calculation
